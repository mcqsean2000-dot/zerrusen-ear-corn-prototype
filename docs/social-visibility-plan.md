# Theo's Farm Social Visibility Plan

This plan supports Theo's Farm visibility by creating steady social posts that point people back to `https://theosfarm.com`. Social activity should support search discovery, but it does not replace the main SEO work on the website itself.

## Current Setup

- Codex automation: `Theo's Farm Weekly Social Post Batch`
- Schedule: Mondays at 8:30 AM
- Output: seven draft posts for Facebook and Instagram, suggested hashtags, and suggested images
- Review file: `docs/social-post-drafts.md`
- Publishing: manual review and posting for now
- Goal: build consistent brand and keyword signals around Theo's Farm whole ear corn

## Posting Rules

- Link back to `https://theosfarm.com` in every post.
- Use natural language first; avoid keyword stuffing.
- Rotate topics so the feed does not sound repetitive.
- Keep claims grounded in approved client information.
- Use real product or packing photos when possible.
- Prefer practical buyer language: wildlife feeders, backyard feeding, squirrel corn, deer corn, whole ear corn, cob corn.
